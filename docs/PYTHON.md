# Python 


## Array
- Definition: ```x=[]``` 
- Iteration: ```for index, value in enumerate(x)```
- Length: ```len(x)```
- Insert: ```x.insert(index,item) ```
- Sort: ```x.sort()```

## String
- Definition: ```s=””```
- Convert: ```str()```
- Trim/remove leading and trailing spaces: ```s.strip()```
- Slice = [start:stop:step]  ```(reverse [::-1])```

## Dictionary
- Definiton: ```d={key:value, key:value,... }```
