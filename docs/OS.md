# Operating Systems

### Processing Types
- **Single State**: only one security level at a time
- **Multi State**: multiple states at one time

### Processing modes
- **privileged, kernel, system, supervisor**:
  - full privileges
- **problem state, user**:
  - limited privileges
  - CPU only allows a subset of full instruction set
